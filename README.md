## Installation

WIToast is available through [CocoaPods](https://cocoapods.org). To install
it, simply add the following line to your Podfile:

```ruby
pod 'WIToast'
```

```ruby
pod 'WIToast', :git=> 'https://github.com/huipengo/WIToast.git'
```
